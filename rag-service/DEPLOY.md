## Step 4 â€” Verify it works

Your service URL will look like:

`https://eif-rag-service.onrender.com`

Test health:

```bash

```

Expected: `{"status":"ok","service":"rag-service"}`

Trigger initial DNA indexing (use your `RAG_SERVICE_SECRET` from Render env):

```bash
curl -X POST "https://eif-rag-service.onrender.com/reindex?force=true" \
  -H "X-RAG-Secret: YOUR_RAG_SERVICE_SECRET"
```

## Troubleshooting

| Problem | Fix |
|---------|-----|
| Deploy fails on build | Check Render logs; usually a missing env var or pip error |
| `/health` times out | Free tier is waking up â€” wait 60s and retry |
| `/reindex` returns 401 | `X-RAG-Secret` header must match `RAG_SERVICE_SECRET` in Render |
| `/reindex` fails on Docs | Ensure `GOOGLE_SERVICE_ACCOUNT_JSON` is valid one-line JSON |
| Retrieval empty | Run migration SQL first, then `/reindex?force=true` |
